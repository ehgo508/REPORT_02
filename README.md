# top — 프로세스 모니터링
> `top`은 **현재 시스템의 프로세스 및 메모리 사용 현황을 실시간으로 출력**하는 명령어입니다.  
기본적으로 5초 간격으로 업데이트되며, 다음 정보가 표시됩니다.

## 주요 특징
- 시스템 자원(CPU, MEM 등)을 실시간으로 확인 가능
- 프로세스 상태(Running, Sleeping 등) 확인
- ps 명령어와 달리 정보가 자동 갱신됨
- 서버 모니터링 시 가장 많이 사용되는 명령어 중 하나

## top 단축키 정리

| 명령어 | 설명 |
|--------|-------|
| `space` | 정보를 업데이트 |
| `^L` | 화면 초기화 |
| `F`, `f` | 출력 필드 추가/제거 |
| `O`, `o` | 출력 필드 정렬 순서 변경 |
| `H`, `h` | 명령어 도움말 출력 |
| `k` | 프로세스 종료 |
| `n`, `#` | 출력 프로세스 수 지정 |
| `s` | 업데이트 주기 변경 |
| `w` | 설정을 ~/.toprc에 저장 |
| `q` | 종료 |

## 화면 예시
```
top - 14:25:10 up 10 days,  2:33,  2 users,  load average: 0.32, 0.40, 0.44
Tasks: 200 total,   1 running, 199 sleeping,   0 stopped,   0 zombie
%Cpu(s):  3.5 us,  1.0 sy,  0.0 ni, 95.0 id,  0.3 wa,  0.0 hi,  0.2 si,  0.0 st
```

# ps
> `ps`는 **프로세스의 현재 상태를 출력하는 명령어**입니다.  
PID, TTY, TIME, CMD 같은 프로세스 정보를 조회할 수 있습니다.

## 주요 특징
- top은 실시간, ps는 현재 상태를 “한 번” 출력
- PID, UID, PPID 등 중요한 프로세스 정보 확인 가능
- grep과 함께 특정 프로세스를 찾는 데 자주 사용됨

## ps 단축키 정리

| 옵션 | 설명 |
|------|------|
| `-A` | 모든 프로세스 출력 |
| `-N` | `-A`와 동일하나 ps 제외 |
| `-a` | 세션 리더 및 터미널 없는 프로세스 제외 |
| `-d` | 세션 리더 제외 모든 프로세스 |
| `-e` | 커널 프로세스 제외 |
| `T` | 현재 터미널 모든 프로세스 |
| `a` | 현재 터미널 사용자 프로세스 |
| `r` | 실행 중 프로세스 |
| `x` | 터미널 없는 프로세스 출력 |
| `--deselect` | `-N`과 동일 |

## 출력 예시
```
UID   PID  PPID  C STIME TTY          TIME CMD
root    1     0  0  Feb07 ?        00:01:34 /sbin/init
user 2341  1234  0  13:25 pts/0    00:00:00 bash
```

# jobs 
> `jobs`는 **백그라운드 작업 상태를 출력**하는 명령어입니다.

## 특징
- `&` 로 실행한 명령을 관리함
- 중지된(stopped) 작업도 표시됨
- fg/bg 명령과 함께 자주 사용됨

## jobs 상태 종류

| 상태 | 설명 |
|------|------|
| **Running** | 작업 실행 중 |
| **Done** | 정상 종료(0 반환) |
| **Done(code)** | 오류 종료 (0 아닌 값 반환) |
| **Stopped** | 작업이 일시 중단됨 |
| **Stopped(SIGTSTP)** | SIGTSTP 신호에 의해 중단됨 |
| **Stopped(SIGSTOP)** | SIGSTOP 신호에 의해 중단됨 |
| **Stopped(SIGTTIN)** | SIGTTIN 신호에 의해 중단됨 |
| **Stopped(SIGTTOU)** | SIGTTOU 신호에 의해 중단됨 |

## 예시
```bash
$ sleep 100 &
[1] 5321

$ jobs
[1]+  Running  sleep 100 &
```

# kill
> `kill`은 **프로세스를 종료시키기 위해 시그널을 전달**하는 명령어입니다.

## 주요 신호
| 번호 | 이름 | 설명 |
|------|--------|-----------------------------|
| **9** | SIGKILL | 즉시 강제 종료 |
| **15** | SIGTERM | 정상 종료 요청 (기본) |
| **2** | SIGINT | Ctrl + C |
| **18** | SIGCONT | 중지된 프로세스 재개 |
| **19** | SIGSTOP | 프로세스 중지 |


## 사용 예시
```bash
kill 1234       # PID 1234 종료
kill -9 1234    # 강제 종료
kill %1         # jobs에서 작업 번호 1 종료
```
