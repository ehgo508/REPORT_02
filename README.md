# top — 프로세스 모니터링
> 실시간으로 시스템의 CPU/메모리 사용량과 프로세스 정보를 보여주는 명령어

## 주요 특징
- 시스템 자원(CPU, MEM 등)을 실시간으로 확인 가능
- 프로세스 상태(Running, Sleeping 등) 확인
- ps 명령어와 달리 정보가 자동 갱신됨
- 서버 모니터링 시 가장 많이 사용되는 명령어 중 하나

## 주요 옵션
```bash
top             # 기본 실행
top -d 1        # 1초 주기로 갱신
top -u 사용자명  # 특정 사용자의 프로세스만 보기
top -p PID      # 특정 PID만 모니터링
```

## 화면 예시
```
top - 14:25:10 up 10 days,  2:33,  2 users,  load average: 0.32, 0.40, 0.44
Tasks: 200 total,   1 running, 199 sleeping,   0 stopped,   0 zombie
%Cpu(s):  3.5 us,  1.0 sy,  0.0 ni, 95.0 id,  0.3 wa,  0.0 hi,  0.2 si,  0.0 st
```

# ps
> 실행 중인 프로세스를 스냅샷 형태로 출력하는 명령어

## 주요 특징
- top은 실시간, ps는 현재 상태를 “한 번” 출력
- PID, UID, PPID 등 중요한 프로세스 정보 확인 가능
- grep과 함께 특정 프로세스를 찾는 데 자주 사용됨

## 주요 옵션
```bash
ps            # 현재 셸에서 실행 중인 프로세스
ps -e         # 모든 프로세스 출력
ps -f         # Full-format 출력
ps aux        # BSD 스타일(가장 일반적으로 사용)
ps -ef        # 리눅스 표준 스타일
```

## 출력 예시
```
UID   PID  PPID  C STIME TTY          TIME CMD
root    1     0  0  Feb07 ?        00:01:34 /sbin/init
user 2341  1234  0  13:25 pts/0    00:00:00 bash
```

# jobs 
> 현재 셸에서 실행 중인 백그라운드 작업 목록을 보여주는 명령어

## 특징
- `&` 로 실행한 명령을 관리함
- 중지된(stopped) 작업도 표시됨
- fg/bg 명령과 함께 자주 사용됨

## 주요 명령
```bash
jobs             # 백그라운드 작업 목록 확인
bg %1            # 작업 번호 1을 백그라운드에서 재개
fg %1            # 작업 번호 1을 포그라운드로 가져오기
```

## 예시
```bash
$ sleep 100 &
[1] 5321

$ jobs
[1]+  Running  sleep 100 &
```

# kill — 프로세스 종료
> 프로세스에 특정 신호(signal)를 보내 종료·중지·재개하는 명령어

## 주요 신호
| 번호 | 이름 | 설명 |
|------|--------|-----------------------------|
| **9** | SIGKILL | 즉시 강제 종료 |
| **15** | SIGTERM | 정상 종료 요청 (기본) |
| **2** | SIGINT | Ctrl + C |
| **18** | SIGCONT | 중지된 프로세스 재개 |
| **19** | SIGSTOP | 프로세스 중지 |


## 사용 예시
```bash
kill 1234       # PID 1234 종료
kill -9 1234    # 강제 종료
kill %1         # jobs에서 작업 번호 1 종료
```

# 명령어 비교 테이블

| 명령어 | 기능 | 실시간 여부 | 대표 옵션 | 설명 |
|--------|------|-------------|------------|----------------------------|
| **top** | 실시간 프로세스 모니터링 | O | -d, -u | CPU/MEM 모니터링 |
| **ps** | 프로세스 상태 출력 (스냅샷) | X | -ef, aux | 실행 중 프로세스 조회 |
| **jobs** | 백그라운드 작업 목록 | X | 없음 | 셸 작업 제어 |
| **kill** | 프로세스 종료 | X | -9, -15 | 신호를 보내 종료 |
